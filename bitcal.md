# 简单位运算

---

### 位运算的概念

计算机内部的所有数据都是以**二进制**的形式存储的。位运算，顾名思义，就是直接对整数在内存中的**二进制位**进行操作。

位运算的执行速度非常快，在算法竞赛和 GESP 考试中，我们经常使用位运算来代替一些普通的算术运算，从而优化程序的时间复杂度，或者解决特定的逻辑问题。

---

### 常见的位运算符与运算规则详解

在编写 C++ 代码时，最常见的位运算符有 $6$ 个。为了让大家更直观地看清“位”的变化，下面的例子我们统一用 **$8$ 位二进制数（$8\ \text{bit}$）** 来做演示。

#### 1. 按位与（&）
**运算规则**：两个对应的二进制位**都为 1 时，结果才为 1**，否则为 0。（口诀：全 1 为 1，有 0 即 0）
**举例**：计算 $10 \& 12$
十进制的 $10$ 转为二进制是 $00001010$
十进制的 $12$ 转为二进制是 $00001100$
$$
\begin{align*}
&00001010 \quad (10)\\
\& \quad &00001100 \quad (12)\\
\hline
&00001000 \quad (8)
\end{align*}
$$

**结果**：十进制的 **$8$** 。

---

#### 2. 按位或（|）
**运算规则**：两个对应的二进制位**只要有一个为 1，结果就为 1**。（口诀：有 1 为 1，全 0 才 0）
**举例**：计算 $10 | 12$
$$
\begin{align*}
&00001010 \quad (10)\\
| \quad &00001100 \quad (12)\\
\hline
&00001110 \quad (14)
\end{align*}
$$

**结果**：十进制的 **$14$** 。

---

#### 3. 按位异或（^）
**运算规则**：两个对应的二进制位**相同为 0，不同为 1**。（口诀：找不同，消消乐）
**举例**：计算 10 \^ 12
$$
\begin{align*}
&00001010 \quad (10)\\
\char94 \quad &00001100 \quad (12)\\
\hline
&00000110 \quad (6)
\end{align*}
$$

**结果**：十进制的 **$6$** 。

---

#### 4. 按位取反（~）
**运算规则**：这是一个单目运算符，将每个二进制位的 **0 变为 1，1 变为 0**。
**举例**：计算 $\sim 10$
$$
\begin{align*}
\sim \quad &00001010 \quad (10)\\
\hline
&11110101 \quad (245)
\end{align*}
$$

> **注意**：在真实的 C++ 环境中，`int` 类型通常是 32 位且带符号的。如果直接输出 `~10`，结果会是 `-11`。这是因为计算机使用了**补码**来表示负数。这里为了让大家看懂翻转的过程，我们暂时把它当成无符号的 8 位数字来理解。

---

#### 5. 左移（<<）
**运算规则**：将所有二进制位整体向左移动若干位，左边超出的高位丢弃，右边空出的低位**补 0**。
**举例**：计算 $10 << 2$（将 $10$ 左移 $2$ 位）
* 原数字：$00001010$ $(10)$
* 左移后：$00101000$ $(40)$
**妙用**：在二进制中，**左移相当于乘 2**。$a << k$ 相当于 $a \times 2^k$。这里 $10$ 左移 $2$ 位，等于 $10 \times 2^2 = 40$。

---

#### 6. 右移（>>）
**运算规则**：将所有二进制位整体向右移动若干位，右边超出的低位丢弃。
**举例**：计算 $10 >> 1$（将 $10$ 右移 $1$ 位）
* 原数字：$00001010$ $(10)$
* 右移后：$00000101$ $(5)$
**妙用**：在二进制中，**右移相当于除以 2 并向下取整**。$a >> k$ 相当于 $a \div 2^k$。这里 $10$ 右移 $1$ 位，等于 $10 \div 2^1 = 5$。

---

### 实战练习

利用位运算判断奇偶性。
在以前，我们判断一个数 $n$ 是否为奇数，通常使用 $n \% 2 \neq 0$。现在我们可以用位运算来实现更高效的判断。

**思考**：所有偶数的二进制最低位一定是 $0$，所有奇数的二进制最低位一定是 $1$。所以，我们只需要把 $n$ 和数字 $1$ 进行**按位与（\&）**运算，如果结果是 $1$ 就是奇数，是 $0$ 就是偶数。

#### 输入格式
一个正整数 $n$，范围在 $[1, 1000000]$ 之间。

#### 输出格式
如果是奇数输出 $\text{Odd}$，偶数输出 $\text{Even}$。

输入 $5$ 输出 $\text{Odd}$
输入 $10$ 输出 $\text{Even}$
输入 $1024$ 输出 $\text{Even}$

---

```c++
#include <iostream>
using namespace std;
/*
(1) n & 1 可以快速提取 n 的二进制最低位
(2) 位运算符的优先级通常低于比较运算符（如 ==），所以 (n & 1) 必须要加括号！
*/

int main()
{
    int n;
    cin >> n;
    
    if ((n & 1) == 1) 
    {
        cout << "Odd" << endl;
    }
    else 
    {
        cout << "Even" << endl;
    }
    
    return 0;
}
```

### 拓展技巧：异或交换律

利用异或（^）可以在不借助第三个变量的情况下，直接交换两个变量的值。这是一个非常经典的小技巧。

**原理**：任何数和自己异或都是 $0$（$a \oplus a = 0$），任何数和 $0$ 异或都是自己（$a \oplus 0 = a$）。

```c++
int a = 3, b = 5;
a = a ^ b;
b = a ^ b;
a = a ^ b; // 此时 a 变成了 5，b 变成了 3
```
